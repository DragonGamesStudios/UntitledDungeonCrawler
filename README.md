# UntitledDungeonCrawler

## Installation

To install Untitled Dungeon Crawler (UDC), you'll first need [git](https://git-scm.com/) and [CMake](https://cmake.org).

1. Clone the repository

```
git clone https://github.com/DragonGamesStudios/UntitledDungeonCrawler
```

2. Build with CMake

```
cd UntitledDungeonCrawler
mkdir build
cmake ..
```

3. Install using your favourite compiler

Example: makefile

```
make
or
make install
```
